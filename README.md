# Java Practice Code

A collection of Java programming exercises and fundamentals practice.

## Contents

- Basic Java syntax and operators
- Object-oriented programming concepts
- Control flow and conditional statements
- Data structures and algorithms practice
- Exception handling
- File I/O operations

## Tech Stack

Java 23

## Getting Started

```bash
git clone https://github.com/JohnImorin/Java-practice-code.git
cd Java-practice-code
```

Compile and run:
```bash
javac Program.java
java Program
```
